프로젝트 목적

Univeristy Ranking 프로젝트의 목적은 사용자에게 대학을 보기 쉽게 순위별로 나열해 보여주는 것입니다. 국내 대학 순위나 학과별 대학 순위 정보를 찾기 어려운 점을 해결하고자 해외 웹사이트 2 곳 (QS Univeristy Ranking, Times Higher Education)에서 국내 대학 데이터를 가져와 보기 편리하게 만들었습니다.

Univeristy Ranking 프로젝트는 현재 2페이지로 구성되어 있습니다 (1. 메인페이지, 2. 상세페이지).

메인페이지는 대학교를 순위별로 나열해주는 페이지입니다. 필터 기능을 사용해서 원하는 학교를 검색하거나 학과별로 순위를 재나열 할 수 있습니다. 또한, 데이터를 2곳에서 가져왔기에 QS 또는 THE 버튼을 눌러 사이트별 대학 순위가 어떻게 달라지는지 볼 수 있습니다.

상세페이지는 원하는 대학교에 대한 정보가 들어있는 페이지로, 연도별 순위 변동 그래프, 학교 소개, 경쟁률, 학생 교수 비율, 총 학생 수, 학비, 그리고 해당 대학교의 학과별 순위가 나와 있습니다.

Contribution

Univeristiy Ranking 프로젝트는 Vue3와 Spring Boot를 사용하고 있습니다. Vue를 사용한 프론트엔드는 제가 담당하고 Spring Boot를 사용한 백엔드는 다른 팀원이 담당합니다. 현재는 프로젝트가 두 페이지밖에 없지만 추후 로그인 및 회원가입 기능, 마이페이지 등등을 추가할 예정입니다.

웹사이트 디자인과 기능 구현은 제가, 데이터 처리 관련 작업은 팀원이 맡았습니다. 신뢰할 수 있는 대학 순위 정보를 가져오기 위해 제가 웹크롤링을 따로 배워 2개의 웹사이트에서 데이터를 수집했습니다.

Improvement

로고 제작을 시작으로 웹사이트의 모든 부분을 직접 디자인했지만, 역시 처음이다 보니 어색한 부분도 있고 허전한 부분도 있었습니다. 현재 도메인을 구매한 상태이기에 프로젝트를 올린 후 사용자의 피드백을 받으며 하나하나 고쳐나갈 생각입니다.

React가 현재 Vue보다 많이 사용되고 있어서 React 강의를 새로 수강하고 있습니다. 강의가 끝나면 React를 이용하여 다른 프로젝트를 구현해 볼 생각입니다.
